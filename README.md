# Getting started

Clone this repository to htdocs dir.

```bash
  git clone *clone link*
```

Go to project directory 

```bash
  cd Lekcje-Svelte-2021
```

Install packages using yarn / npm

```bash
  yarn 
  
  npm install
```

Import MySql DB, it is located in root directory as DB.sql


Start project

```bash
  yarn dev
```
